## 📝 테스트 절차

### 1️⃣ **브라우저 강력 새로고침**
- `Ctrl + Shift + R` (Windows/Linux)
- `Cmd + Shift + R` (Mac)

### 2️⃣ **로그인**
```

비밀번호: test1234
```

### 3️⃣ **기능 테스트**

#### **기업절세계산기**
1. 회사명, 지역, 직원 수 입력
2. 전년도 법인세, 세율 입력
3. 사내근로복지기금 출연금 입력
4. "계산하기" 클릭
5. 결과 확인:
   - 법인세 절감액
   - 사내근로복지기금 효과
   - 연간 총 절세액

#### **CEO절세계산기**
1. 회사 기본 정보 입력
2. 주식 정보 입력
3. 증여/상속 시뮬레이션
4. 결과 확인:
   - 주식 평가액
   - 증여세/상속세
   - 절세 전략

#### **직원절세계산기**
1. 직원 급여 정보 입력
2. 사내근로복지기금 활용
3. 절세 효과 계산

#### **네트급여계산기**
1. 목표 실수령액 입력
2. 세전 급여 자동 계산
3. 세금 상세 분석

#### **기업리스크진단**
1. 진단 문항 응답
2. AI 분석 리포트 생성
3. 맞춤형 솔루션 확인
