## 🔧 테스트 시나리오

### 1. 기업회원 가입 테스트
1. 샌드박스 URL 접속
2. "회원가입" 탭 클릭
3. "기업회원" 선택
4. 폼 작성:
   - 회사명: (테스트) 주식회사 ABC
   - 기업회원분류: 법인 선택
   - 추천인: 홍길동 (← 중요! 컨설턴트 시트에 있어야 함)
   - 이름: 김철수
   - 전화번호: 010-1234-5678
   - 이메일: test@company.com
   - 비밀번호: test1234
   - 비밀번호 확인: test1234
5. "회원가입" 버튼 클릭
6. Google Sheets '기업회원' 시트에 데이터 추가 확인
7. 승인여부를 "승인완료"로 수동 변경

### 2. 로그인 테스트
1. 전화번호: 010-1234-5678
2. 비밀번호: test1234
3. 로그인 성공 시 메인 앱으로 이동
4. Google Sheets에서 로그기록 확인

### 3. 컨설턴트 로그인 테스트
1. "컨설턴트" 탭 선택
2. 전화번호: 010-8765-4321
3. 비밀번호: 12345
4. 로그인 성공 확인

## 💡 중요 사항

1. **추천인 검증**
   - 기업회원 가입 시 추천인은 필수 입력 항목
   - 추천인 이름은 '사근복컨설턴트' 시트의 A열(이름)과 정확히 일치해야 함
   - 추천인의 승인여부가 "승인완료"여야 가입 가능

2. **승인 시스템**
   - 회원가입 후 자동으로 "대기중" 상태
   - 관리자가 Google Sheets에서 수동으로 "승인완료"로 변경
   - "승인완료" 상태만 로그인 가능

3. **로그 기록**
   - 모든 활동이 Google Sheets J열에 자동 기록됨
   - 날짜/시간 포함
   - 관리자가 사용자 활동 추적 가능
