## 🔧 **방법 2: Application Storage 완전 삭제**

일반 모드에서 계속하려면:

```
1. 개발자 도구 (F12)

2. Application 탭 클릭

3. 좌측 메뉴에서 "Storage" 확장

4. "Clear site data" 클릭

5. 다음 항목 모두 체크:
   ✅ Local storage
   ✅ Session storage
   ✅ IndexedDB
   ✅ Web SQL
   ✅ Cookies
   ✅ Cache storage
   ✅ Application cache

6. "Clear site data" 버튼 클릭

7. 페이지 완전 새로고침:
   Ctrl + Shift + R
```
