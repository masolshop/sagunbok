## 🎯 핵심 기능

### 1. 시크릿 VIP 디자인
```
🔐 SECRET PLAN 배지
🌟 암버/오렌지 그라데이션
✨ 프리미엄 멘트 느낌
🎨 다크 히어로 섹션
```

### 2. 인터랙티브 체크리스트
```
5가지 리스크 항목:
✓ 유보금(잉여금)이 계속 쌓인다
✓ 가지급금이 정리되지 않는다
✓ 자사주/차명주식 이슈가 있다
✓ 가업승계를 '언젠가' 해야 한다
✓ 통상임금/노무 리스크가 불안하다

위험도 평가:
- 2개 이상 체크 → 🚨 고위험 (즉시 컨설팅 필요)
- 1개 체크 → ⚠️ 주의 (사전 점검 권장)
- 0개 체크 → 정상
```

### 3. 4가지 SECRET PLAN
```
📈 미처분이익잉여금 PLAN
   쌓일수록 주식가치↑ → 증여·상속세 폭증

💳 가지급금 PLAN
   인정이자·상여처분 → 세무조사 트리거

👑 가업승계 PLAN
   평가방식/타이밍/구조에 따라 세금이 갈림

🎯 절세 PLAN
   법인세 대표종소세 절감 + 복지적금 설계로 리스크를 '비용'으로 전환
```

### 4. 담당자 자동 표시 로직
```typescript
// 우선순위 1: 현재 로그인한 사용자가 매니저/컨설턴트
if (currentUser.userType === 'manager' || 'consultant') {
  return 본인 정보 (이름, 직함, 전화번호)
}

// 우선순위 2: 기업회원인 경우 추천인 정보
if (currentUser.userType === 'company' && currentUser.referrer) {
  return 추천인 정보
}

// 우선순위 3: 기본 담당자
return 이종근 단장 (01063529091)
```
